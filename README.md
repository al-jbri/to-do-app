# ✅ Al-jbri To-Do

A lightweight, open-source educational web application built entirely with vanilla web technologies (no frameworks). Designed for short-term task management, it focuses on a simplified priority system and delivers a fully responsive experience across all devices.

## ✨ Features

* **Built with Core Web Tech:** Weighing in at under 300KB, the app offers unprecedented loading speeds. Powered by Vanilla JavaScript for dynamic, seamless DOM updates without any page reloads.
* **100% Local Storage:** Your data never leaves your device. Everything is stored locally in your browser's `localStorage`, ensuring maximum speed, offline availability, and complete privacy.
* **Modern UI:** Features a clean aesthetic with rounded corners, a carefully selected harmonious color palette, and a soothing dark theme designed to reduce eye strain.
* **Fully Responsive:** Whether you're opening the app on a smartphone, tablet, laptop, or a massive desktop monitor, the layout adapts perfectly (featuring smooth snap-scrolling on smaller screens).
* **Open Source:** Use it directly from the live site, clone it locally for instant offline access, or fork the repository to customize and build your own version for the community.

## 🪄 Getting Started

### Live Demo
You can start using the app instantly by visiting the official website: **[aljbri-todo.netlify.app](https://aljbri-todo.netlify.app)**

### Run Locally
For the fastest offline access, you can run the app directly on your machine without needing a server:

1. Open your favorite terminal in your desired directory.
2. Clone the repository by running:
```bash
git clone https://github.com/al-jbri/to-do-app
```
3. Navigate to the project folder and simply open the `index.html` file in your browser. That's it!
